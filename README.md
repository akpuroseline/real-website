# Akpu Roseline Portfolio (MadameRose)

A high-end, editorial-style portfolio website built for a Web Developer, Web3 Copywriter, and Automation Engineer. This project features a "Brutalist-Luxury" aesthetic using a monochromatic color palette and sophisticated typography.

## 🚀 Features

- **Custom Interactive Cursor**: A unique SVG arrow cursor that replaces the default pointer and reacts to interactive elements (links, buttons, inputs).
- **Scroll Progress Indicator**: A sleek progress bar at the top of the viewport indicating reading progress.
- **Animated Text Ticker**: A continuous marquee showcasing key roles and locations, reinforcing the editorial brand identity.
- **Role Cycling Animation**: Dynamic text in the About section that cycles through your core specialties: Web Development, Web3 Copywriting, and Automation Engineering.
- **Reveal on Scroll**: Smooth entrance animations for sections and cards implemented via the Intersection Observer API for better performance.
- **Interactive Service Cards**: Cards featuring subtle parallax movement and hover effects to increase user engagement.
- **Booking Modal & Contact Form**: A custom-built modal interface with form validation and [EmailJS](https://www.emailjs.com/) integration for direct client inquiries.
- **Fully Responsive**: A mobile-first design approach with a custom hamburger menu and optimized layouts for tablets and smartphones.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (utilizing CSS Variables, Grid, and Flexbox), and Vanilla JavaScript.
- **Typography**: 
    - *Bricolage Grotesque* (Body)
    - *DM Serif Display* (Headings)
    - *Space Mono* (Metadata/Mono)
- **Integrations**: EmailJS for serverless contact form functionality.

## 📂 Project Structure

- `index.html`: The core structure of the portfolio, including the hero, about, services, portfolio, and contact sections.
- `style.css`: The "Editorial Black & White" design system, including animations and responsive media queries.
- `script.js`: The engine for all interactive components, cursor logic, scroll effects, and form handling.

## 🔧 Setup

1. Clone the repository.
2. Update the `EMAILJS_PUBLIC_KEY`, `EMAILJS_SERVICE_ID`, and `EMAILJS_TEMPLATE_ID` in `script.js` with your own credentials.
3. Open `index.html` in any modern web browser.
